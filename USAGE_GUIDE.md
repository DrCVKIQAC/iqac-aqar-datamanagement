# 📚 AQAR Management System - Usage Guide

## How to Use Every Feature

---

## 🔐 Login

### Step 1: Open the System
- Click the AQAR link or open `AQAR_OER_Complete.html`
- You'll see the login page

### Step 2: Select Your Role
Click one of 8 roles:
- **IQAC Coordinator** — Monitor and consolidate all data
- **Principal** — View completion status
- **Department Head** — Submit department data
- **Admission In-Charge** — Submit admission statistics
- **Examination In-Charge** — Submit exam results
- **Scholarship Section** — Submit scholarship data
- **NSS In-Charge** — Submit NSS activities
- **NCC In-Charge** — Submit NCC activities

### Step 3: Enter Your Name
Type your full name in the text field

### Step 4: Select Department (if applicable)
Choose your department from dropdown

### Step 5: Click Login
You're now logged in!

---

## 📊 IQAC Coordinator Dashboard

### What You See
A real-time dashboard with:
- Submission counts
- Status of all departments & sections
- Timeline of submissions
- Auto-consolidated data

### Available Actions

#### 1. Monitor Submissions
- **Dashboard Tab**: See all submissions in real-time
- **Departments Card**: Shows how many departments submitted
- **Status Cards**: Shows status of Admission, Exam, Scholarship
- **Timeline Table**: Lists all submissions and timestamps

#### 2. View Auto-Consolidated Data
- **Consolidated Tab**: See total data across all departments
- **Auto-Summed**: Research papers, students, placements = all automatic
- **Updated Live**: Changes as new submissions arrive

#### 3. Generate Reports
- **Reports Tab**: Export AQAR data
- **Download CSV**: Get data in spreadsheet format
- **Download PDF**: Generate full report (when implemented)

#### 4. Send Reminders
- **Pending List**: See what's still missing
- **Take Action**: Contact departments for missing data
- **Automatic Alerts**: Know instantly who hasn't submitted

### Real-Time Monitoring
Dashboard updates every 5 seconds. You see:
- When each department submits
- Exactly what data they submitted
- When they submitted
- Overall completion percentage

## 👥 Department Head Usage

### What to Submit
1. **Total Students**: How many students in your department
2. **Pass Percentage**: Success rate (0-100)

## 💾 Data Management

### Backup Your Data
1. Regularly download your submissions
2. Store copies in cloud (Google Drive, Dropbox)
3. Keep multiple backups
4. Recommended: Weekly backups

### How to Download
1. As IQAC, click "Download AQAR"
2. File saves as AQAR_Data.json
3. Store safely

### How to Restore
- Keep backup files for reference
- You can always resubmit if needed

## 🔒 Privacy & Security

### Your Data is Private
- Stored only in your browser
- Not sent to external servers
- Only IQAC can see all data
- Department heads see only their data

### No Cloud Services
- No third-party servers
- Complete institutional control
- FERPA compliant
- Offline capable

### Best Practices
1. Don't share login details
2. Log out when finished
3. Clear browser cache periodically
4. Use strong passwords
5. Don't share device with others

## 🆘 Troubleshooting

### Data Not Saving
1. Check internet connection
2. Allow localStorage in browser settings
3. Try different browser
4. Clear browser cache
5. Disable browser extensions

### Can't Login
1. Check you selected a role
2. Check you entered your name
3. Try different browser
4. Clear cache and cookies

### Dashboard Not Updating
1. Click "Refresh Dashboard" button
2. Wait 5 seconds (auto-refresh)
3. Reload page (F5 or Cmd+R)
4. Try different browser

### Data Lost
1. Check localStorage isn't cleared
2. Try downloading your backup
3. Resubmit the data
4. Contact IQAC for help
