## 🎯 One-Page Summary

**AQAR Management System** is a free, open-source web app for automatic AQAR data collection.

**What It Does:**
- 8 user roles submit data
- System auto-consolidates
- IQAC sees everything
- Generates AQAR report

**How to Get Started:**
1. Download HTML file
2. Open in browser
