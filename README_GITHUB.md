## 🚀 What is This?

**AQAR Management System** is a completely free, open-source web application that helps educational institutions automate Annual Quality Assurance Report (AQAR) data collection, consolidation, and reporting.

**This is an Open Educational Resource (OER)** licensed under CC BY 4.0. You can use it, modify it, and share it freely with other colleges and universities!

## 📊 How It Works

### Workflow in 30 Seconds

```
1. IQAC sends link to all department heads & sections
                        ↓
2. Each submits their data simultaneously:
   • Dept Heads (5 min each)
   • Admission Section (5 min)
   • Examination Section (5 min)
   • Scholarship Section (5 min)
   • NSS Coordinator (5 min)
   • NCC Coordinator (5 min)
                        ↓
3. IQAC logs in and sees EVERYTHING auto-consolidated
   (No manual data entry needed!)
                        ↓
4. IQAC downloads complete AQAR as CSV
                        ↓
5. Submits to university ✓
```

**Total IQAC time: ~10 minutes!**
